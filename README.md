## 接口测试平台项目

- 编程语言: JavaScript
- 构建工具: Vite 4.x
- 前端框架: Vue 3.x + setup
- 路由工具: Vue Router 4.x
- 状态管理 : Pinia
- UI 框架: Arco Design
- CSS 预编译: Less
- HTTP 工具: Axios
- Git Hook 工具 : Husky
- 代码规范: EditorConfig + Prettier + ESLint
- 提交规范: Commitizen + Commitlint

## 安装使用

- 安装依赖

```shell
cd fox-client
npm install
```

- 运行

```shell
npm run dev
```

- 预览

```shell
npm run preview
```

- 打包

```shell
npm run build
```

- 提交 commit
  - 方式一: `git cz`
  - 方式二：`npm run commit`
